# Features

  - Upload file (image, audio, video, document)
  - Auto generate multi size for image

### Installation

Requires to run:
> + PHP 5.6+
> + Laravel 5.4+


Install the dependencies by composer

```sh
$ composer require softoneasia/laravelupload
```

Publish config file

```sh
$ php artisan vendor:publish --tag=config
```
